# Code Of Conduct

Sonoic is meant to be a calm, constructive place to collaborate.

## Expected Behavior

- Be respectful and patient
- Assume good intent
- Give feedback on ideas and code without making it personal
- Help keep discussions clear, practical, and welcoming
- Leave room for people to ask basic questions without embarrassment

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Dismissive or hostile communication
- Discrimination of any kind
- Deliberate disruption of discussions or contribution workflows

## Project Standard

We want the same tone in collaboration that we want in the codebase:

- clear
- direct
- thoughtful
- not overcomplicated

If something feels off, open an issue or contact the maintainer so we can address it early and calmly.
