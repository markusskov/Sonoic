# Product Specs Index

The current product source of truth is split across:

- [README.md](../../README.md) for public product direction and setup.
- [plan.md](../../plan.md) for roadmap, current state, open questions, and architecture guardrails.
- [Manual Host Refresh Verification](../manual-host-refresh-verification.md) for the fallback host and room refresh validation checklist.

## Current MVP

Sonoic is focused on fast control for the active Sonos target:

- reliable play, pause, next, previous, and scrubbing
- fast volume and mute access
- honest stale or unavailable state
- clear now-playing metadata and artwork
- practical room, queue, Home, Apple Music, and home-theater flows once each slice proves its value

## Spec Rule

When a feature becomes more detailed than `README.md` or `plan.md` can comfortably hold, add a focused spec in this directory and link it from this index.
